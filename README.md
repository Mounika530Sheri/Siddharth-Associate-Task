# Trade Analysis Project

This project analyzes import/export trade data for household and kitchen products.  
It includes automatic processing of the sample Excel file, HSN code analysis,  
year-wise and month-wise summaries, and export of charts.

## Files Included
- process_trade_data.py — Main Python script
- Sample Data 2.xlsx — Raw data
- output_charts/ — Generated charts
- Trade_Analysis_YourName.xlsx — Final Excel analysis output

## How to Run
1. Install Python 3 and packages:
   pip install pandas matplotlib openpyxl

2. Run:
   python process_trade_data.py

All outputs are stored in the output_charts folder.
